# 文本對齊工具 (Text Alignment Tool)

## 📝 專案介紹
這是一個專業的文本對齊工具，主要功能包括：
- 字幕與文本同步對齊
- 音頻處理
- Word 文檔比對
- 多語言支持（繁體中文）

## ✨ 主要特性
- 精確的文本同步
- 支持多種音頻格式
- 智能字幕編輯
- 友好的使用者介面

## 🛠 系統需求
- Python 3.8+
- Windows / macOS / Linux
